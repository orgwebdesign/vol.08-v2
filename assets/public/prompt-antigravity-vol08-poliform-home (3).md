# Prompt Antigravity — VOL.08 Home Page HTML/CSS

## Objectif
Créer une **page Home premium** pour **VOL.08**, une plateforme B2B fermée de distribution d’ameublement & décoration sourcée au Maroc.

Le style doit s’inspirer fortement du site Poliform Casablanca fourni : design très minimal, blanc/noir, grandes images, grille éditoriale, typographie propre, beaucoup d’espace blanc, sections calmes et luxueuses.

## Important
- Générer uniquement une page Home.
- Utiliser uniquement **HTML + CSS + JavaScript vanilla léger si nécessaire**.
- Pas de framework.
- Créer les fichiers :
  - `index.html`
  - `style.css`
  - `script.js` si besoin
- Design responsive desktop/tablette/mobile.
- Style premium, sobre, architectural.
- Couleurs : blanc, noir, gris clair, gris moyen.
- Pas de couleurs bleues/oranges du sitemap dans le design final.
- Ne pas copier Poliform à 100%, mais garder la même sensation : luxe calme, catalogue premium, grande photographie, grille nette.

## Assets
Utiliser le logo fourni : `logo Vol.08.svg`

Si les images ne sont pas disponibles, utiliser des placeholders propres depuis Unsplash avec thème :
- mobilier premium
- décoration intérieure
- showroom
- artisanat marocain moderne
- matières naturelles

## Style CSS attendu

## Direction visuelle
- Grandes marges.
- Grille max-width 1440px.
- Espacement section : 100px desktop.
- Images avec object-fit cover.
- Cards sans arrondis ou très léger radius 2px.
- Boutons simples : noir/blanc/outline.
- Hover discret : opacity, underline, image zoom léger.
- Pas d’effets flashy.
- Pas de gradients colorés.
- Pas de design SaaS bleu.
- Pas de grosses icônes modernes.
- Le rendu doit être proche d’un site premium mobilier / architecture.

## Responsive
Mobile :
- menu devient burger simple
- hero en une colonne
- grille produits en une colonne
- sections avec padding réduit
- topbar simplifiée
