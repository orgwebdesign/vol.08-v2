# Prompt — Recréer la plateforme VOL.08

## Prompt FR

```text
Crée une plateforme B2B privée pour une entreprise de distribution d'ameublement et décoration haut de gamme au Maroc.

Stack : HTML5 / CSS3 / JavaScript (Vanilla) — pas de framework, pas de build tool.

Couleurs :
- Fond : #1c1c16
- Surface/cartes : #2a2a24
- Texte : #f0f0ea
- Texte secondaire : #8a8a82
- Bordures : #3a3a34
- Accent doré : #c9a96e
- Accent clair : #e8d5a3

Typographie : Grift font (ou équivalent → font style "expanded", graisses 400 à 700)

Icons : Lucide (https://unpkg.com/lucide@latest)

Pages à créer :

1. index.html — Page de connexion élégante, sombre, minimaliste. Logo centré, formulaire email + mot de passe, lien mot de passe oublié + inscription. Côté droit : phrase d'accroche + témoignage.

2. dashboard.html — Dashboard client. Sidebar gauche avec navigation (Dashboard, Catalogue, Favoris, Devis, Commandes, Compte). Topbar avec recherche + notifications + déconnexion. Stats : commandes, devis en cours, favoris, en attente. Tableaux des dernières commandes et devis en attente.

3. catalogue.html — Grille responsive de produits. Filtres par catégorie, collection, prix. Chaque carte : image, référence, nom, catégorie, prix. Pagination.

4. produit.html — Fiche produit : images, nom, référence, prix, description, fiches techniques, bouton ajouter aux favoris / demander un devis.

5. selections.html — Favoris et sélections avec grille et vue liste.

6. devis.html — Panier et création de devis.

7. commandes.html — Liste des commandes avec statuts.

8. compte.html — Mon compte : profil, utilisateurs, préférences.

9. notifications.html — Centre de notifications.

10. support.html — Assistance prioritaire.

11. inspirations.html — Moodboard.

12. inscription.html — Formulaire d'inscription professionnelle (nom, email, société, statut, pays, téléphone).

13. mot-de-passe-oublie.html — Réinitialisation de mot de passe.

14. commercial/dashboard.html — Dashboard commercial.

15. commercial/index.html — Portefeuille clients.

16. admin/index.html — Administration dashboard.

Design system :
- Boutons : fond #c9a96e, texte foncé, border-radius 6px
- Sidebar : fond #1c1c16, largeur 240px, items avec icônes
- Topbar : fond #2a2a24, avec barre de recherche
- Cards : fond #2a2a24, border-radius 10px, box-shadow subtil
- Tables : pleine largeur, fond alterné, badges de statut (succès vert, warning jaune, info bleu)
- Mobile : bottom navigation, menu hamburger
- Animations : transitions douces sur hover, transform scale subtil sur les cartes
```

## Prompt EN

```text
Create a private B2B platform for a high-end furniture and decor distribution company in Morocco.

Stack: HTML5 / CSS3 / JavaScript (Vanilla) — no framework, no build tool.

Colors:
- Background: #1c1c16
- Surface/cards: #2a2a24
- Text: #f0f0ea
- Secondary text: #8a8a82
- Borders: #3a3a34
- Gold accent: #c9a96e
- Light accent: #e8d5a3

Typography: Grift font (or equivalent → expanded style, weights 400 to 700)

Icons: Lucide (https://unpkg.com/lucide@latest)

Pages to create:

1. index.html — Elegant, dark, minimalist login page. Centered logo, email + password form, forgot password link, signup link. Right side: tagline + testimonial.

2. dashboard.html — Client dashboard. Left sidebar with navigation (Dashboard, Catalog, Favorites, Quotes, Orders, Account). Topbar with search + notifications + logout. Stats: orders, pending quotes, favorites, awaiting. Tables for recent orders and pending quotes.

3. catalogue.html — Responsive product grid. Filters by category, collection, price. Each card: image, reference, name, category, price. Pagination.

4. produit.html — Product detail: images, name, reference, price, description, technical specs, add to favorites / request quote buttons.

5. selections.html — Favorites and selections with grid and list views.

6. devis.html — Cart and quote creation.

7. commandes.html — Order list with statuses.

8. compte.html — My account: profile, users, preferences.

9. notifications.html — Notification center.

10. support.html — Priority support.

11. inspirations.html — Moodboard.

12. inscription.html — Professional registration form (name, email, company, status, country, phone).

13. mot-de-passe-oublie.html — Password reset page.

14. commercial/dashboard.html — Commercial dashboard.

15. commercial/index.html — Client portfolio.

16. admin/index.html — Admin dashboard.

Design system:
- Buttons: background #c9a96e, dark text, border-radius 6px
- Sidebar: background #1c1c16, width 240px, items with icons
- Topbar: background #2a2a24, with search bar
- Cards: background #2a2a24, border-radius 10px, subtle box-shadow
- Tables: full width, alternating rows, status badges (green success, yellow warning, blue info)
- Mobile: bottom navigation bar, hamburger menu
- Animations: smooth hover transitions, subtle scale transform on cards
```
